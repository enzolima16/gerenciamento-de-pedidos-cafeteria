# Sistema de Gerenciamento: Cafeteria

### Desafio DIO - Design Patterns com Java
**Tecnologia:** Java 21, Spring Boot 3.x, Maven, H2 Database

Repositório com a implementação de um sistema de gerenciamento destinado a uma cafeteria, o qual explora Design Patterns abordados no Lab "Explorando Padrões de Projetos na Prática com Java". O projeto, por meio do Spring Framework, utiliza padrões como:
 - Strategy/Repository
 - Facade
